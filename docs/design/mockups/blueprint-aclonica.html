What this is. First pass at the blueprint / Aclonica direction. Real Google Fonts (Aclonica, Work Sans, Architects Daughter, Caveat, Courier Prime), CSS-rendered hatching and dashes, inline SVG for compass + scale bar. The map screen is the showpiece — drafting-paper for everything else.

What's interesting. Flights are "plates" and zones are drafted boundaries. The title block becomes the universal page header on any record (flight detail, zone detail, eventually conservation reports). The cyanotype-on-map / drafting-paper-everywhere-else split makes the map feel like a plan-being-drafted while the rest feels like the drafting table itself.

What to react to. Is Aclonica too bold? Does "Super·Intendent" with the middle dot work or should it be unbroken? Is the drafted-red accent (#8B2C1E) right or do you want a different ink color? Should the map be even more drafted (more hand-drawn annotations, more bunker hatching, contour lines) or kept this restrained?
